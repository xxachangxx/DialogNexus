## [0.1.0-dev] - 2025-01-21

### 🚧 开发中
### ⚠ BREAKING CHANGES
- 项目从 Vite 迁移至 Next.js 框架，不兼容旧版本

### ✨ 新增
- 基础界面框架搭建完成
  - 主体聊天界面布局
  - 消息输入区域
  - 消息显示区域
  - 基础响应式设计

### 🎯 功能
- 实现基础消息功能
  - 消息发送功能
  - 消息编辑功能
  - 消息清除功能
  - 确认对话框

### 🛠 开发体验
- 集成 TypeScript 支持
- 集成 shadcn/ui 组件库
- 集成 Tailwind CSS 样式框架
- 添加快捷键支持
  - Ctrl + Enter 发送消息
  - Ctrl + L 清除消息
